Entity Framework The Entity Framework Team has released a new release candidate for the Entity Framework. The new release candidate has three new features: LINQ to Entities, LINQ to SQL, and LINQ to Entities Dynamic Query now compile all queries as expression trees, which improves performance. The Entity Framework is included in the “Microsoft ASP.NET MVC 2 Tools Update,” which is an extension for Visual Studio 2008 that includes templates and other extensions for ASP.NET MVC 2 RC 1. You can get more information on this extension at http://aspnetmvc.codeplex.com/. As you can see, LINQ to Entities is an alternative to Linq to SQL. LINQ to SQL has not changed.

When it comes to remote, it's hard to get some good suggestions because there are so many types of remotes to deal with. If you have the HDTV Remote app in your Android, it will recognize the signal from your TV, but I'm not sure how to make it work in MyRemote. The problem with this app is it was designed for the PS3, which has a very different remote. The best solution for this is to call the manufacturer's customer service and they can help you install a new set of drivers. If you already have a PC, the easiest way is to hook it up to your TV using the VGA cable. Then go to Add > Hardware > Remotes > HDMI > Sony. This will make the computer recognize the new remote in the video settings and add the function to your menu. This is probably the easiest method.

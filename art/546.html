We've recently discovered a new method to simplify our work in this area, while at the same time provide significant new benefits to our global customers. As a result of this change, both the description and the sample of the new attribute will change on August 29th. We'll continue to describe the same general attribute, but with different levels of detail in our documents. On December 31st 2001, we will begin using the new "tag" to replace our descriptions. Customers will see the same attribute names on the web, in EDI, in documentation, and in data, all with the same meanings. The "definition tag" and tag number will appear as part of our XML schema. The attributes will now have a fixed, simplified sample, helping us resolve problems when that data isn't used. It will no longer be used to define more specific data or sub-attributes. The simplified attribute values will eliminate complex dates and times, and "date formats" which were previously used to define attributes that need to be uniquely defined for each individual company. Instead, the same simplified attribute will apply to all customers. The following list contains the simplified attributes (currently scheduled to be implemented on December 31st 2001) and the attributes they will replace. All simplified attributes will follow this form: List of Simplified Attributes Each bushel by weight. Each ton by weight. Each short ton by weight. Each million pounds. Each hundredweight by weight. Each metric ton. Each short metric ton. Each barrel by weight. Each count per piece. Each pound per piece. Each yard by weight. Each piece per yard. Dry corrected each bushel by weight. While this change may not seem dramatic to some, the resulting simplification and consistency will greatly ease our work. As always, thank you for your business and support.

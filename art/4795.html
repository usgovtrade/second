As the number of computer devices utilized by a given user increases, the amount of digital media stored on, managed by, and accessible to the user devices also increases. Users often find it difficult to collect, organize, and playback all of their digital media to enjoy it. There exist systems that allow users to stream stored multimedia contents from one or more user devices to a media server and stream multimedia contents from the media server to other user devices. These systems, however, can be extremely challenging and difficult to configure and manage for users. Thus, there is a need for an improved system and method for streaming multimedia contents in a media server in a media system. This and other needs are addressed by the present disclosure.

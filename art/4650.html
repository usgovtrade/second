The present invention relates to an image communication apparatus and, more particularly, to an image communication apparatus for communicating images with another apparatus through a telephone line. In a known image communication system for communicating images with another apparatus through a telephone line, it is not possible to perform transmission and reception of images simultaneously. The transmission and reception of images will be described in detail. In the system, a user makes a reservation for an image communication on the basis of a telephone number and a PIN (personal identification number). Thereafter, the user operates a personal computer at a time of image transmission to set image data to be transmitted to an object, e.g., a photo-stamp or an address book. In addition, the user makes a reservation for image reception by making a phone call to a destination of reception. When the image data transmission and the image reception are completed, the user performs a confirmation operation to end the image communication. The image communication apparatus includes a display such as an LCD display or a CRT display for displaying data inputted or stored in the apparatus. In the system, an image data is transmitted to a photo-stamp. If the image data is a still image, the image data is converted into voice data for transmission to a destination of reception. However, in a conventional image communication apparatus, in order to transfer voice data, there is a necessity of a memory for storing voice data, thereby resulting in a large size of the apparatus. In addition, the conventional image communication apparatus has another problem that a user cannot find image data that is stored in the apparatus. A first object of the present invention is to provide an image communication apparatus and method capable of easily communicating images with another apparatus. A second object of the present invention is to provide an image communication apparatus and method capable of performing a file transfer by transferring voice data without a necessity of a memory for storing the voice data. To achieve the above objects, according to the present invention, there is provided an image communication apparatus for communicating image data with another apparatus, comprising: telephone communication means for communicating a telephone number and a PIN to another apparatus; image data input means for inputting image data; registration means for registering a file name of a file to be transmitted and telephone numbers or addresses of a destination of transmission of the file, on the basis of the registered telephone numbers or addresses; file storage means for storing a file in response to a telephone number or address of the destination of transmission, the file inputted by the image data input means, and a telephone number of a transmission destination; and telephone communication means for communicating a telephone number and a PIN to the destination of transmission. The present invention provides another image communication apparatus for communicating image data with another apparatus, comprising: telephone communication means for communicating a telephone number and a PIN to another apparatus; image data input means for inputting image data; telephone communication means for communicating a telephone number and a PIN to a telephone number of a transmission destination; storage means for storing the image data inputted by the image data input means; and telephone communication means for communicating the telephone number and the PIN to the telephone number of the transmission destination.

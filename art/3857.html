The Buddy System can be used for a variety of different functions, from monitoring the environment to recording movement. Each module has two modes of operation, its default and the “Buddy” mode. The default mode allows for the sensors and the sensor network to gather data. In the Buddy mode the information that is being collected is transmitted over the RF link to the companion unit which in turn creates and displays an interface to indicate to the user that data has been collected. The units are able to self configure once configured. Depending on the application for the user, the communication range between the sensor units may vary. In some cases, the units can operate as stand-alone, without the need of a hub. The unit can be battery operated and operates in a wide variety of temperature, moisture and shock/vibration ranges. The units can be secured and stored in packages or housings. Each unit has a unique I.D that can be read by a reader when in proximity. Each unit can be attached to its own data service link, which in turn can be attached to a home location network (HAN) through either wireless or wired connections. Each sensor unit has internal memory for use when there is no data service link. The sensor system is available to commercial customers. The system is modular and expandable. New components can be added to the system using an expansion board that can be attached to the system. The expansion board can also provide other functions such as serial communication to interface with a laptop or the internet. The modularity of the system allows additional functions and sensors to be added as needed, making the system very flexible. Multiple sensor units can be connected to a single base station through wireless connection. The system has internal battery backup so the system is always ready for use. The battery backup is rechargeable through the AC/DC power adapter that also acts as the charging unit for the system. The sensor system uses a sensor network technology to collect data from various components for either a single user or multiple users. The network is able to provide data on any component or environmental parameters. The sensor network technology has a standard communication protocol which enables the sensor units to be connected in different ways to create sensor networks and allow communications between sensor units. Various sensor units can be integrated with a single hub using the same communication protocol. The sensor network protocols are based on various communications standards, such as Bluetooth and RF, and allow connection to other hardware devices such as phones, webcams and laptops through cables and wireless connections. The system will be able to connect to a sensor hub and other sensors or other devices as a part of its communication network. The module uses the Bluetooth protocol for data transmissions. The sensor units will automatically discover all other sensor units within the network and communicate to establish the network and then will transmit data. In some cases, the sensor hub may be part of the sensor network. It will be connected through one of the nodes in the network. The system uses Bluetooth Technology to monitor locations, such as rooms, rooms of a house, or the outside environment for multiple users, or to monitor and compare environmental data between multiple users. The system uses sensor technology to capture information about any parameter or environmental condition. Examples include air temperature, relative humidity, door position, location, time, and battery level. Additional sensors may include other environmental parameters, such as sound and motion. The sensor system is also able to collect video, audio and take still image data. In an exemplary embodiment, the sensor system is able to receive data from various sensors and input information from the communication network through the RF link. The sensor units will transmit any data, such as temperature, relative humidity and battery level. The sensor units will also be able to transmit a video or audio message. In some cases, the hub may be connected to other electronic devices such as a laptop or computer.

In jQuery you can write a similar code: $(function() { $('.active_div a').click(function(e) { e.preventDefault(); $(this).parent().addClass('inactive_div'); $(this).parent().removeClass('active_div'); }); }); $('.inactive_div').click(function(e) { e.preventDefault(); $(this).parent().addClass('active_div'); $(this).parent().removeClass('inactive_div'); }); Note that you can use the same style elements above into your document.

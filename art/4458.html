The present disclosure relates generally to a method and system for providing wireless data service to a mobile computing device. With the proliferation of wireless data devices and increasing mobile computing, there is an ever increasing need for users to have access to the data network from any location. Since data networks are becoming more congested, it is becoming more and more difficult to deliver sufficient data rates to each user that is connected to the data network. Thus, it would be desirable for a method and system that overcomes these identified deficiencies. The present disclosure provides such a method and system.
